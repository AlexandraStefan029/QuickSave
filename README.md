The Chrome Application is an MVP  browser extension designed to improve user productivity and enhance the browsing experience. The extension provides a lightweight and fast solution that runs directly in Google Chrome.

The main goal of the MVP is to validate core functionality with minimal features while ensuring a smooth and intuitive user experience. The application interacts with web pages through Chrome APIs, allowing it to perform actions such as reading page content, storing user data, or displaying quick tools directly in the browser.

This version focuses on essential features only, in order to test usability and gather feedback for future improvements and additional functionality.

The project was developed using HTML, CSS, and JavaScript, along with Chrome Extension APIs.
